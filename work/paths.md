# Paths

Only: `work/{team}/{space}/{project-name}/`

Deliverable: one `README.md` per project folder. Single file per project; do not create other files (notes, drafts, separate docs). Add new problems, topics, or findings to that README. Never create a new folder or new README for new problems or topics.

Only the team, space, and project-name folder names listed in the tree below are allowed by default. If the system thinks new folders need to be created for work that are not in this tree, ask the user to name them; the system may suggest names.

- work
 - team
  - space
   - project-name

If a level is not listed, ask before creating the path.

Format: "Only" line unchanged; tree one indent per depth.
